The big picture

Mental model

The model never touches your filesystem. It can't. It asks; your handler runs; the bytes come back as data. Hold onto that asymmetry — it's both the security model and the programming model.

Gotcha

failureMode defaults to "error". For almost every tool you'll write, that default is wrong. Set it to "return" alongside any non-Never failure schema.

Rule of thumb

If your section is a flow, use flow-diagram. If it's a checklist, use step-through. If you want the reader to feel time, use scrub-timeline.
